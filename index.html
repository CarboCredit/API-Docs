CarboCredit API Documentation

Empowering sustainable development through API integration.

Base URL

https://handy-seahorse-suddenly.ngrok-free.app/api/

Endpoints

Get All Projects

Endpoint: /get_projects

Method: GET

Description: Retrieve a list of all available projects.

Responses:

  • 200 OK: Returns a list of projects.
  • 404 Not Found: No projects found.
  • 500 Internal Server Error: An error occurred on the server.

Get Project by Name

Endpoint: /project/:project

Method: GET

Description: Retrieve information about a specific project based on its name.

Parameters:

  • project (string): The name of the project.

Responses:

  • 200 OK: Returns the project's information.
  • 404 Not Found: Project not found.
  • 500 Internal Server Error: An error occurred on the server.

Error Handling

All endpoints return appropriate HTTP status codes and error messages in case of issues.

Logging

Each request is logged with its method and URL for debugging and auditing purposes.
